# Generate Agent 4 Data Assets

The `.asset` instances for progression data are generated by:

- Editor menu: `MetalPod/Progression/Generate Agent 4 Data Assets`
- Script: `Assets/Scripts/Progression/Editor/Agent4DataAssetGenerator.cs`

Run this once in Unity after scripts compile. It creates/updates:

- `DefaultHovercraftStats.asset`
- `GameConfig.asset`
- All course, environment, upgrade, cosmetic, and hazard assets under this folder.
